GidaLink — Lifestyle Image Placeholders
=========================================

URLs so the site looks complete out of the box. Replace them with real,
downloaded images for a more authentic feel. Each placeholder is marked
with a comment in app/page.tsx showing exactly where it's used.

Download images for these 4 spots and save them in THIS folder
(public/images/) using the exact filenames below. The code currently
points to Unsplash URLs — once you've downloaded your own images, swap
the <img src="..."> values in app/page.tsx to "/images/<filename>".
